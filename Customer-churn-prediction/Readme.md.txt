# Customer Churn Prediction

## Problem Statement
Customer churn is a major issue for subscription-based businesses. The goal of this project is to predict whether a customer is likely to churn based on historical data.

## Dataset
Telco Customer Churn Dataset

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Approach
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model building using Logistic Regression and Random Forest
- Model evaluation using accuracy and confusion matrix

## Results
The Random Forest model performed better compared to Logistic Regression and provided useful insights for reducing customer churn.

## Conclusion
This project helps businesses identify high-risk customers and take proactive retention measures.
